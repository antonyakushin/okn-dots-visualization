Settings

Movement:

This is the direction that stripes will move across the screen.

Number of stripes:

This is the total number of stripes that appear on the screen at any time.

Speed:

This is the number of seconds it takes a stripe to go from one side of the screen to the other.

Stop after:

This is the number of seconds after which the application should automatically exit and return to the settings screen.

Colors:

This is the color of the main and alternating stripes.

Run in fullscreen:

Fullscreen will take up the whole screen. To exit fullscreen mode, press the escape key.

Usage

To exit the application and return to the settings, click anywhere on the screen or press any key.
